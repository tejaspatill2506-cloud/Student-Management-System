# Student-Management-System
A Python-based Student Management System that performs CRUD operations using Object-Oriented Programming, File Handling, and Exception Handling to efficiently manage student records.

The Student Management System is a console-based Python application designed to simplify the management of student information. It allows users to add, update, delete, search, and display student records while storing data permanently using file handling or SQLite. The project demonstrates Python programming fundamentals, Object-Oriented Programming (OOP), CRUD operations, modular programming, and exception handling.

Features
Add Student Records
Update Student Information
Delete Student Records
Search Student Details
Display All Students
File Handling / SQLite Storage
Input Validation
Exception Handling
